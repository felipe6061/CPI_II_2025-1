Questões sobre Objetos em JavaScript

  1. O que é um objeto em JavaScript?
    Um objeto em JavaScript é uma coleção de propriedades, onde cada propriedade é uma associação entre um nome (chave) e um valor. Pode conter variáveis (chamadas de propriedades) e funções (chamadas de métodos). Exemplo representando uma pessoa: 
    const pessoa = {
    nome: "Maria",
    idade: 30,
    profissao: "Engenheira"
};
  2. Quais são os tipos de valores que uma propriedade de objeto pode conter?
    Uma propriedade pode conter qualquer tipo de dado válido em JavaScript: 
    const exemplos = {
    string: "Texto",      // String
    numero: 42,          // Número
    booleano: true,      // Booleano
    array: [1, 2, 3],    // Array
    objeto: {            // Outro objeto
        chave: "valor"
    },
    funcao: function() { // Função
        console.log("Olá");
    },
    nulo: null,          // Null
    indefinido: undefined // Undefined
};
  3. Diferença entre propriedades e métodos em um objeto

    Propriedades são características do objeto (armazenam valores). Métodos são ações que o objeto pode realizar (funções).

    const carro = {
    // Propriedades
    marca: "Ford",
    ano: 2020,
    
    // Método
    ligar: function() {
        console.log("Carro ligado");
    }
};
  4. Adicionar método saudar ao objeto aluno 
    const aluno = {
    nome: "João",
    idade: 20,
    aprovado: true,
    saudar: function() {
        console.log("Olá, meu nome é " + this.nome);
    }
};
  5. O que são objetos aninhados?

    São objetos dentro de outros objetos, usados para organizar informações complexas. Exemplo:

    const pessoa = {
    nome: "Carlos",
    endereco: {
        rua: "Rua das Flores",
        numero: 123,
        cidade: "São Paulo"
    }
};
  6. Compare objetos e arrays

    Arrays são usados para listas ordenadas de itens (acessados por índice numérico). Objetos são usados para coleções não ordenadas de pares chave-valor, quando os itens têm nomes significativos.

    Use array para listas simples (como uma lista de compras) e objeto para representar entidades com propriedades (como um usuário com nome, email, etc.).

  7. Acessar propriedades com for...in 
    const carro = {
    marca: "Toyota",
    modelo: "Corolla",
    ano: 2022
};

for (let propriedade in carro) {
    console.log(propriedade + ": " + carro[propriedade]);
}
    Saída: 
    marca: Toyota
modelo: Corolla
ano: 2022
  8. O que é uma função construtora?

    É uma função usada para criar objetos com a mesma estrutura. Começa com letra maiúscula por convenção, para diferenciar de funções normais.

    function Carro(marca, modelo) {
    this.marca = marca;
    this.modelo = modelo;
}
  9. Função construtora Livro 
    function Livro(titulo, autor) {
    this.titulo = titulo;
    this.autor = autor;
    this.exibirInfo = function() {
        console.log(`Título: ${this.titulo}, Autor: ${this.autor}`);
    }
}

const meuLivro = new Livro("Dom Casmurro", "Machado de Assis");
meuLivro.exibirInfo();
  10. O que acontece ao usar new Pessoa("Maria", 25)?

    Quando usamos new, acontece:

    1. Um novo objeto vazio é criado
    2. A função é executada com this apontando para o novo objeto
    3. As propriedades e métodos são adicionados ao objeto
    4. O objeto é retornado automaticamente (não precisa de return)

    No exemplo, cria um objeto Pessoa com nome "Maria", idade 25 e um método saudar().
