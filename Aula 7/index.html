Questões sobre DOM em JavaScript

  1. O que é o DOM e qual o seu papel na manipulação de páginas web com JavaScript?

    O DOM (Document Object Model) é uma representação em forma de árvore de todos os elementos de uma página HTML. Ele permite que o JavaScript:

    • Acesse e modifique elementos HTML
    • Altere estilos CSS
    • Adicione ou remova elementos
    • Responda a eventos (cliques, teclas, etc.)

    O DOM conecta o HTML com o JavaScript, tornando as páginas dinâmicas e interativas.

  2. Qual a diferença entre getElementById() e querySelector()?

    getElementById() seleciona apenas por ID, enquanto querySelector() usa seletores CSS (mais versátil). Exemplos:

    // getElementById - seleciona apenas por ID
let elemento = document.getElementById('meuId');

// querySelector - pode usar qualquer seletor CSS
let elemento = document.querySelector('#meuId'); // Por ID
let elemento = document.querySelector('.minhaClasse'); // Por classe
let elemento = document.querySelector('div p'); // Por hierarquia
  3. O que é HTMLCollection e como ela se comporta?

    HTMLCollection é uma lista viva de elementos retornada por métodos como getElementsByClassName(). Ela se atualiza automaticamente quando novos elementos com a mesma classe são adicionados ao DOM.

    let itens = document.getElementsByClassName('item');
console.log(itens.length); // Mostra 2 itens inicialmente

// Adiciona novo item dinamicamente
let novoItem = document.createElement('div');
novoItem.className = 'item';
document.body.appendChild(novoItem);

console.log(itens.length); // Agora mostra 3 itens, sem precisar re-selecionar
  4. Manipulando lista de itens

    Código para selecionar e alterar a cor dos itens:

    let itens = document.querySelectorAll('.item');

itens.forEach(item => {
    item.style.color = 'blue';
});
  5. Diferença entre innerHTML e textContent

    innerHTML retorna/define todo o conteúdo HTML, incluindo tags. textContent retorna/define apenas o texto, ignorando tags.

    Use innerHTML quando precisar incluir tags HTML. Use textContent quando quiser apenas o texto (mais seguro contra XSS).

    let div = document.querySelector('div');

// textContent (seguro)
div.textContent = '<script>código malicioso</script>'; // Mostra como texto

// innerHTML (permite HTML)
div.innerHTML = '<strong>Negrito</strong>'; // Renderiza o negrito
  6. Alterando atributos de imagem

    Função para modificar a imagem:

    function alterarImagem() {
    let foto = document.getElementById('foto');
    foto.src = 'imagem2.jpg';
    foto.alt = 'Nova imagem';
    foto.style.width = '300px';
}
  7. Adicionando parágrafo

    Código completo:

    let div = document.getElementById('conteudo');
let novoParagrafo = document.createElement('p');
novoParagrafo.textContent = 'Bem-vindo!';
div.appendChild(novoParagrafo);
  8. Métodos para navegar no DOM
    • parentNode - Acessa o elemento pai
    • children - Retorna os elementos filhos (não inclui textos)
    • nextElementSibling - Acessa o próximo irmão
    • previousElementSibling - Acessa o irmão anterior
    let item = document.querySelector('li');
let pai = item.parentNode; // Pai (ul)
let primeiroFilho = pai.children[0]; // Primeiro li
let proximo = item.nextElementSibling; // Próximo li
  9. O que será exibido no console?

    Será exibido: "Olá, mundo!"

    textContent pega todo o texto dentro do elemento, incluindo o texto dentro de elementos filhos, mas sem as tags HTML.

  10. Desafio prático: Adicionar novo item

    Solução:

    function adicionarItem() {
    let lista = document.getElementById('minhaLista');
    let novoItem = document.createElement('li');
    novoItem.textContent = 'Novo item';
    lista.appendChild(novoItem);
}
