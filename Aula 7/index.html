Questões sobre DOM em JavaScript

  1. O que é o DOM e para que ele serve?

    O DOM (Document Object Model) é como um "mapa" de todos os elementos da página. Ele organiza todos os elementos (títulos, parágrafos, imagens) numa estrutura clara que o JavaScript consegue entender e modificar.

    • Encontre e mude partes do HTML
    • Altere cores, tamanhos e outros estilos
    • Adicione ou remova coisas da página
    • Reaja a cliques e outras ações do usuário

    É como um tradutor que ajuda o JavaScript a entender e modificar a página.

  2. Qual a diferença entre getElementById() e querySelector()?

    getElementById() só busca por ID, enquanto querySelector() é mais esperto e aceita vários tipos de busca:

    // getElementById - só funciona com ID
let elemento = document.getElementById('meuId');

// querySelector - busca de vários jeitos
let elemento = document.querySelector('#meuId');  // Por ID
let elemento = document.querySelector('.classe'); // Por classe
let elemento = document.querySelector('div p');   // Por relação entre tags
  3. O que é HTMLCollection?

    É uma lista de elementos que se atualiza sozinha quando a página muda. Por exemplo:

    let itens = document.getElementsByClassName('item');
console.log(itens.length); // Mostra 2 itens

// Adiciona um item novo
let novoItem = document.createElement('div');
novoItem.className = 'item';
document.body.appendChild(novoItem);

console.log(itens.length); // Agora mostra 3, sem precisar buscar de novo

    Diferente de arrays normais, essa lista muda automaticamente.

  4. Como mudar a cor de vários itens de uma vez? 
    let itens = document.querySelectorAll('.item');

itens.forEach(item => {
    item.style.color = 'blue'; // Muda cor para azul
});
  5. Qual a diferença entre innerHTML e textContent?

    innerHTML entende tags HTML, textContent só trabalha com texto puro.

    Quando usar cada um:

    let div = document.querySelector('div');

// textContent - mais seguro
div.textContent = '<script>código perigoso</script>'; // Aparece como texto

// innerHTML - entende HTML
div.innerHTML = '<strong>Texto em negrito</strong>'; // Realmente fica em negrito

    Dica: Use textContent quando não precisar de formatação.

  6. Como trocar uma imagem? 
    function trocarImagem() {
    let foto = document.getElementById('foto');
    foto.src = 'imagem2.jpg';    // Caminho da nova imagem
    foto.alt = 'Foto nova';      // Texto alternativo
    foto.style.width = '300px';  // Largura
}
  7. Como adicionar um parágrafo novo? 
    let div = document.getElementById('conteudo');
let paragrafoNovo = document.createElement('p');
paragrafoNovo.textContent = 'Olá!';  // Texto do parágrafo
div.appendChild(paragrafoNovo);       // Adiciona na página
  8. Como encontrar elementos próximos?

    Você pode navegar pelo "mapa" do DOM assim:

    • parentNode - Elemento pai
    • children - Elementos filhos diretos
    • nextElementSibling - Próximo irmão
    • previousElementSibling - Irmão anterior
    let item = document.querySelector('li');
let pai = item.parentNode;          // Pai (a tag ul)
let primeiroFilho = pai.children[0]; // Primeiro item
let proximo = item.nextElementSibling; // Próximo item da lista
  9. O que aparece no console?

    Aparece: "Olá, mundo!"

    Porque textContent pega todo o texto, inclusive dentro de outras tags, mas sem as tags em si.
